# UAS-2
Website kuis online adalah sebuah aplikasi berbasis web yang digunakan untuk menyajikan soal kuis kepada pengguna secara interaktif. Pengguna dapat mengerjakan soal pilihan ganda dalam batas waktu tertentu, kemudian sistem akan menilai jawaban secara otomatis dan menampilkan hasil skor setelah kuis selesai.
Alur logika/algoritma
1. Aplikasi quiz dijalankan oleh pengguna.
2. Sistem menampilkan halaman utama quiz yang berisi judul, soal, pilihan jawaban, skor, dan timer.
3. Sistem menampilkan soal pertama beserta beberapa pilihan jawaban.
4. Pengguna memilih salah satu jawaban yang tersedia.
5. Sistem memeriksa jawaban yang dipilih pengguna.
6. Jika jawaban benar, sistem menambah skor pengguna.
7. Jika jawaban salah, skor tidak bertambah.
8. Sistem menampilkan soal berikutnya.
9. Langkah 4 sampai 8 diulangi hingga seluruh soal selesai.
10. Setelah semua soal dijawab, sistem menampilkan skor akhir pengguna.
11. Aplikasi quiz selesai.
